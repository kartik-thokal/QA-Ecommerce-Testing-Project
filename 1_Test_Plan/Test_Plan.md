# Test Plan – E-commerce Application

## Objective
To verify core functionalities like login, product selection, and cart.

## Scope
- Login functionality
- Product listing
- Add to cart
- Checkout process

## Testing Types
- Functional Testing
- Regression Testing
- Smoke Testing

## Tools Used
- Postman
- Selenium (Node.js)
- SQL

## Out of Scope
- Payment gateway testing
- Performance testing
